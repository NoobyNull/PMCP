Privacy Policy

Last updated: December 2024
Privacy First: PerfectMCP is designed with privacy in mind. Your data stays on your infrastructure.

1. Information We Collect

PerfectMCP operates as a self-hosted solution. The following data is processed locally on your infrastructure:

Data Processed Locally
  • Code and Documents: Source code, documentation, and files you upload for analysis
  • Session Data: Conversation history and context for AI interactions
  • Configuration Data: Server settings, API keys, and user preferences
  • Usage Logs: Server access logs and performance metrics

2. How We Use Your Information

Your data is used exclusively for:

  • Providing AI-powered development assistance
  • Maintaining session context and memory
  • Improving code analysis and suggestions
  • System monitoring and performance optimization

3. Data Storage and Security

Local Storage
  • All data is stored locally on your infrastructure
  • No data is transmitted to external servers (except API calls you configure)
  • You maintain full control over your data
Security Measures
  • Encrypted database connections
  • Secure API key storage
  • Access control and authentication
  • Regular security updates

4. Third-Party Services

PerfectMCP may integrate with third-party AI services based on your configuration:

AI API Providers
  • OpenAI: Code analysis and improvement (if configured)
  • Other AI Services: As configured by you
Important: When using external AI services, your code may be sent to these providers. Review their privacy policies and ensure compliance with your organization's data policies.

5. Data Retention

  • Session data is retained based on your configuration
  • Logs are rotated according to your settings
  • You can delete data at any time through the admin interface
  • No automatic data transmission to external parties

6. Your Rights

As a self-hosted solution, you have complete control over your data:

  • Access: Full access to all stored data
  • Modification: Ability to modify or update any data
  • Deletion: Complete control over data deletion
  • Portability: Export data in standard formats

7. Updates to This Policy

This privacy policy may be updated with new versions of PerfectMCP. Check the documentation for the latest version.

8. Contact Information

For questions about this privacy policy or data handling:

  • Email: privacy@perfectmcp.com
  • GitHub: Report Issues
  • Documentation: Setup Guide
Self-Hosted Advantage: Since PerfectMCP runs on your infrastructure, you maintain complete control and ownership of your data.
